Commands

Keyboard

Command/Control + Shift + Space → Toggle add task

Command/Control + Shift + n → Adds a quick task with "To be edited" title and focuses it while bluring the rest

Command/Control + Shift + j → Opens Dev Tools for all open windows

Command/Control + Shift + e → Opens edit window for the active task (if any)

Command/Control + Shift + k → Opens window with kenban

Command/Control + Shift + c → Opens completed tasks window

Command/Control + Shift + s → Focuses the bar and allows typing in an input the index of a task to focus

Mouse

Left click (on a task) → toggles the task as active/inactive

Middle click (on a task) → completes a task

Right click → opens the context menus - see below

Context menus (right click menu)

Main bar's

Select screen → it can be at the top or bottom of any connected screen

Toggle timer → this has more of a legacy meaning, it toggles the countdown of the pomodoro timer even if the play button is pressed

Completed tasks → toggles the completed tasks window

Toggle always on top → this is a poperty of the window where the app can forcefully stay on top of others

Help → toggles the help window

Task's

Edit → toggles the edit window for the selected task

Delete → deletes the task, can't be undone

Complete → completes the task

Category → allows quickly changing the category of the task from a list

User's (on user icon)

Sign In (if user is not logged in) → toggles login window.

Sign Out (if user is logged in) → deletes any information about the user and disconnects the connection to the server
